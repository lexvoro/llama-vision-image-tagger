AI Image Tagger & EXIF Manager

Это продвинутое приложение для автоматического тегирования и управления метаданными изображений. В отличие от оригинального проекта, эта версия фокусируется на **записи данных напрямую в EXIF** изображений и поддерживает **двуязычное тегирование** (русский/английский) с использованием локальной нейросети Llama 3.2 Vision.

Что нового в этой версии

- **Прямая запись в EXIF**: Теги и описания сохраняются не только в базе данных, но и вшиваются в сами файлы (совместимо с Windows и другими просмотрщиками).
- **Двуязычная логика (EN/RU)**: Система генерирует точные английские теги, а затем переводит их на русский язык для максимальной корректности.
- **Оптимизация под GPU**: Настроено для работы с современными картами (например, RTX 5060 Ti) через Ollama.
- **Гибкая настройка**: Возможность выбора количества тегов и языков генерации прямо в интерфейсе.

Основные функции

- **Умное тегирование**: Использование `llama3.2-vision` для анализа объектов, стилей и настроения кадра.
- **Двойной перевод**: Генерация тегов на английском с последующим автоматическим переводом на русский для исключения "галлюцинаций" ИИ.
- **Управление EXIF**:
    - Поле `ImageDescription` для описаний.
    - Поля `XPKeywords` и `XPTitle` для полной интеграции с поиском Windows.
- **Временное хранение**: Все правки сначала собираются в `image_metadata.json`, а затем массово вшиваются в файлы при нажатии "Save All".
- **OCR (Распознавание текста)**: Извлечение текста с изображений и сохранение его в метаданные.

Установка и запуск

1. **Требования**:
    - Python 3.9+
    - Ollama с загруженной моделью `llama3.2-vision`.
2. **Установка зависимостей**:
    
    bash
    
    ```
    pip install -r requirements.txt
    ```
    
    Используйте код с осторожностью.
    
3. **Запуск**:
    
    bash
    
    ```
    python main.py
    ```
    
    Используйте код с осторожностью.
    
    После запуска откройте `http://127.0.0.1:8000` в браузере.

Как это работает (Workflow)

1. **Выбор папки**: Укажите путь к папке с фото. Приложение просканирует `jpg`, `jpeg`, `png` и `webp`.
2. **Настройка**: Выберите нужное количество тегов и отметьте галочкой "RU", если нужны русские теги.
3. **Обработка**:
    - Нажмите **Process**, чтобы нейросеть проанализировала фото.
    - Нейросеть сначала создаст описание и теги на английском, а затем переведет их на русский.
4. **Сохранение**: Нажмите **Save All Metadata**, чтобы навсегда вшить полученные данные в EXIF-поля файлов.

Структура проекта

- `main.py`: Сервер FastAPI, логика управления файлами и запись EXIF.
- `image_processor.py`: Ядро работы с ИИ — генерация описаний, тегов и перевод через Ollama.
- `static/index.html`: Интерфейс на Vue3 и Tailwind CSS.
